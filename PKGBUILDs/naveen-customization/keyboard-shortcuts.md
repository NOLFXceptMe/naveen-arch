# Keyboard Shortcuts

This document lists the custom keyboard shortcuts configured in the XFCE environment for this system.

## Application Shortcuts
| Shortcut | Action | Command |
| :--- | :--- | :--- |
| `<Super> + Space` | Whisker Menu | `xfce4-popup-whiskermenu` |
| `<Ctrl> + <Alt> + T` | Terminal | `ghostty` |
| `<Super> + E` | File Manager | `thunar` |
| `<Super> + R` | App Finder | `xfce4-appfinder -c` |
| `<Alt> + F2` | App Finder (Collapsed) | `xfce4-appfinder --collapsed` |
| `<Ctrl> + <Alt> + L` | Lock Screen | `xflock4` |
| `<Ctrl> + <Alt> + Delete` | Logout Session | `xfce4-session-logout` |
| `<Ctrl> + <Shift> + Escape` | Task Manager | `xfce4-taskmanager` |
| `<Ctrl> + <Alt> + Escape` | Kill Window | `xkill` |
| `<Print>` | Take Screenshot | `xfce4-screenshooter` |
| `<Shift> + Print` | Select Region to Screenshot | `xfce4-screenshooter -r` |
| `<Alt> + Print` | Screenshot Active Window | `xfce4-screenshooter -w` |

## Window Management (XFWM4)
| Shortcut | Action |
| :--- | :--- |
| `<Ctrl> + Q` | **Close Window** |
| `<Super> + Tab` | Cycle Windows |
| `<Super> + F` | Maximize Window |
| `<Super> + D` | Show Desktop |
| `<Super> + Arrow Keys` | **Tile Window** (Left, Right, Up, Down) |
| `<Super> + `` (Grave) | Switch between windows of same application |
| `<Ctrl> + <Alt> + Left/Right` | Switch Workspace |
| `<Ctrl> + <Shift> + <Alt> + L/R` | Move Window to Workspace |
